# 🚀 Running the Enhanced Invitation Generator App

## Quick Start

### 1. Install Dependencies
```bash
cd "d:/New folder/invitation"
npm install
```

### 2. Start the App
```bash
npm start
```

### 3. Run on Device
- **Scan QR code** with Expo Go app on your phone
- **Press 'a'** for Android emulator
- **Press 'i'** for iOS simulator (Mac only)

## Troubleshooting

### App won't start?
```bash
npm start -- --clear
```

### Animations laggy?
- Close other apps
- Use physical device
- Check device performance

### Footer not showing?
- Scroll to bottom
- Check screen height
- Verify Footer import

### Social links not working?
- Check internet connection
- Verify URL format
- Try different browser

## Performance Tips

1. **Use physical device** for best experience
2. **Close background apps** for smooth animations
3. **Update Expo Go** to latest version
4. **Clear cache** if issues persist
